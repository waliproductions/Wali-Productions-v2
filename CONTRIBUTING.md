# CONTRIBUTING

## Purpose

This repository supports the Wali Productions LLC Digital Headquarters project.

All contributions must align with the approved documentation in `docs/00-Foundation`.

## Authority

The Founder retains final approval authority over all changes.

AI assistants, contractors, developers, and contributors may suggest or implement work, but they may not redefine company mission, identity, governance, branding, or business direction.

## Documentation Standard

Documentation should be clear, professional, organized, and consistent with the approved project structure.

Canonical documents require Founder approval before changes are accepted.

## Code Standard

Code should be:

- Clean
- Maintainable
- Secure
- Accessible
- Documented where necessary
- Built for long-term growth

## Branches

Use clear branch names such as:

- docs/foundation
- feature/homepage
- feature/services
- fix/navigation
- refactor/components

## Pull Requests

Pull requests should clearly explain:

- What changed
- Why it changed
- What files were affected
- Whether Founder approval is required

## Artificial Intelligence

AI-generated work must be reviewed before acceptance.

AI may assist with code, documentation, planning, and review, but all final decisions remain with the Founder.

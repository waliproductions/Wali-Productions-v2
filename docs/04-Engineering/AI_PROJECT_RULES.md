# Purpose

This document defines how artificial intelligence systems assist with the Wali Productions LLC Digital Headquarters project.

---

# AI Responsibilities

AI may assist with:

- Documentation
- Software development
- Architecture recommendations
- Code review
- Testing guidance
- Content drafting
- Technical planning

---

# AI Limitations

AI shall not determine:

- Company mission
- Statement of Faith
- Business strategy
- Official branding
- Legal decisions
- Financial decisions
- Founder authority

---

# Documentation Authority

Approved Canonical documentation always takes precedence over AI-generated recommendations.

Current Founder instructions take precedence over all documentation until documentation is formally updated.

---

# Collaboration

AI is an advisory and implementation tool.

Final authority remains with the Founder.
